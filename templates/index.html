Over het project

BikeFix API is een eenvoudige Python Flask API opgezet met een complete DevOps workflow. De applicatie wordt getest, gebuild en als Docker-image automatisch gedeployed via GitHub Actions.

CI/CD Workflow

  1. Code push naar main
  2. GitHub Actions start automatisch
  3. Tests worden uitgevoerd met Pytest
  4. Docker-image wordt gebouwd
  5. Image wordt gepusht naar Docker Hub
